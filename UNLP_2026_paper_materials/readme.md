# Supplementary Materials for the Paper

This repository contains supplementary materials for the research paper on **Automated CEFR-Level Assignment for Ukrainian Texts**.

## Repository Structure

- `figures/` – all visualizations used in the paper (e.g., CEFR distributions, ANOVA results, feature analysis plots)  
- `scripts/` – scripts for data processing, feature extraction, and statistical analysis  
- `results/` – generated outputs from experiments and models  

## Figures

The `figures/` directory contains all graphical materials used in the paper. Figures are organized by analysis type for clarity and reproducibility.

## Scripts

The `scripts/` directory includes scripts for:
- linguistic feature extraction  
- statistical analysis (e.g., ANOVA, regression models)  
- dataset preprocessing  
- evaluation of CEFR classification models  


## Note

This repository is actively maintained. Additional materials and updates may be added as the project evolves.
